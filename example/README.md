# Example

To build and run simply:

     $ ./gradlew clean run

## What is Shown

The basic example is in `example/src/main/java/com/github/nwillc/funjdbc/example/Example.java` and shows:

 - Basic DbAccessor 
 - Extractor use via the factory
 - Enricher use
 - Migrations
 - Query to Stream
 - Finding a single entity
 - Updating
